◆ loadUp()

template<typename data_type >
std::shared_ptr<data_type> ResourceHandler::loadUp ( json::json &  jsonParser,
const std::string &  id 
)

Template loader.

Loads up our data using the appropriate resource holder based on one of the acceptable types. This function is exception safe, as in it returns a nullptr if a exception is given.

Warning
The only acceptable types are: sf::Texture, sf::Font, sf::SoundBuffer and thor::BigTexture.
Template Parameters
data_typeDefines the type of data we will load.
Parameters
jsonParserDefines the json parser that contains our id's file path.
idDefines the key we use to identify our dat in our resource holder.
Returns
Returns std::shared_ptr to our loaded resource.
