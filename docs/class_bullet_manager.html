◆ clearAllBullets()

void BulletManager::clearAllBullets ( )

This method will run through all the bullets and set them to inactive. This will be used on resetting the game to prevent bullets from still being active and on screen when the user restarts the game. This will iterate through the map of bullet vectors and then each bullet in a vector and set their active state to false.
