◆ configure() [1/2]

void GUI::configure ( const Layouts &  layout,
const sf::Vector2u &  windowSize 
)

Configures the GUI layout.

if Custom the layout remains unchanged , if StackVertically the layout takes all non label widgets and lays them out in the x-axis-center and evenly stacked on the y-axis , if StripDiagonal the layout takes all non label widgets and lays them out diagonally stacked

Parameters
layoutdefines the type of layout
windowSizedefines the size of the window.
