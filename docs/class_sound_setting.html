◆ m_attenuation

float SoundSetting::m_attenuation

Attenuation factor of the sound.

The attenuation is a multiplicative factor which makes the sound more or less loud according to it's distance to the listener. A value of 0 makes it so no attenuation occurs. (Defaults to 1)
