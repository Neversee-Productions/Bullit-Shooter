void ai::AiRanged::setState ( std::shared_ptr< ai::states::AiRangedState >  sptrNewState,
bool  rememberPrevious 
)
private

Set a new state.

Assigns our curret state pointer to the new state and pushes that new state onto the stack. If is the passed pointer is nullptr than we remove the state on top of the stack and assign the newest top of the stack to our current state pointer.

Parameters
sptrStatepointer to our new state, nullptr will cause us to use the stack of states, using the design pattern

Pushdown Automata http://gameprogrammingpatterns.com/state.html#pushdown-automata

Parameters
rememberPrevioustells whether or not to add the new state to the stack.

◆ m_stateStack

std::stack<std::shared_ptr<states::AiRangedState> > ai::AiRanged::m_stateStack
private

Describes list of shared pointers to basic states.

Follows the principle of a Finite State Machine, where we instead of switch case'ing through a lot of different states we instead hold a pointer to the current state and merely assign/dereference it. In this case this is a stack of all the current states that this finite state machine is in. Allows for the Finite State Machine to remember the previous state that it was in by push'ing/pop'ing from the stack.
